## 当前定位

Agents-Team 当前是 Public Beta / source-available preview，目标是验证 GitHub-first 的 Goal、PR、测试证据和独立 QA 链路是否能降低 AI coding 长任务返工。当前许可证不是开源许可证；文档中的流程规则不替代真实 CI、分支保护、用户批准或人工判断。

## 反馈原则

Agents-Team 不做静默遥测。反馈应由用户审阅、脱敏，并通过 GitHub Issue 或其他用户明确认可的渠道主动提交。
