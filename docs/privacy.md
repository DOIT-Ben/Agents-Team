# 本地记录与反馈隐私说明

Agents-Team 的运行记录默认只保存在本地，不启用后台遥测，也不会自动上传反馈。日志记录运行编号、版本、阶段、状态和计数等结构化数据，不记录完整提示词、模型输出、源码、环境变量值、密钥、Cookie、邮箱、仓库地址或原始绝对路径。

记录默认保留 **14 天**；单文件达到 20 MiB 后轮转，最多保留 5 个轮转文件。用户可以查看单次运行、预览删除范围，并在明确确认后删除。

反馈流程分为两个确认点：

1. 先生成脱敏预览，不写反馈包，不联网。
2. 用户明确确认后才在本地生成 `feedback.md` 和 `diagnostics.json`。

只有用户另外要求打开 GitHub 时，工具才打开待提交页面；最终提交仍由用户完成。诊断 JSON 不会自动附件。用户应在提交前再次检查，并可以删除任何不希望公开的字段。

发现真实密钥或可利用漏洞时，不要创建公开 Issue，应先删除本地副本并使用 GitHub Private Security Advisory。
